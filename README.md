# Mahjong Game Solving Algorithm

Algorithms and Utilities for judging a Mahjong game's status, including winning hand, ready hand, tile patterns, shanten, and so on. Writen in Rust.
Currently designed for Japanese rule, while other rules may be supported in the future.

## Current Status

Demo. Everything may be changed. No guarantee.
